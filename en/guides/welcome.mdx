## What is the Vendaze API

The Vendaze API is a public REST interface that allows external applications to read and write data in Vendaze, the B2B CRM for sales teams.

With it, you can build native integrations, automations, custom dashboards, and AI agents that operate directly on your commercial team's data.

## How the API works

Vendaze is a **multi-workspace platform**. Each user can belong to multiple workspaces — like a consultant working across several clients. Each workspace is a completely isolated environment with its own data: contacts, deals, tasks, and configurations.

When a user authorizes your app via OAuth, they choose which workspace to connect. All API calls made with that token operate exclusively within that workspace. You never need to send or manage any workspace identifier — the token carries that context automatically.

## What you need before you start

1. A Vendaze account with at least one active workspace
2. A server with HTTPS support to receive the OAuth callback
3. Basic familiarity with REST APIs and the OAuth 2.1 flow

There is no approval process to become an integrator. Any developer can register an app and start building immediately.
