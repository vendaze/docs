O servidor MCP (Model Context Protocol) da Vendaze permite conectar assistentes de IA à sua conta Vendaze. Uma vez conectado, seu assistente pode buscar contatos, visualizar negócios, criar tarefas, registrar atividades e muito mais, tudo dentro da conversa, sem precisar abrir a Vendaze.

## Plataformas com integração nativa

O MCP da Vendaze está disponível como integração listada no **Claude** e no **ChatGPT**. Nessas plataformas, basta buscar por Vendaze e conectar sua conta, sem nenhuma configuração técnica.

## Outras plataformas e IDEs

O MCP da Vendaze pode ser conectado em qualquer cliente que suporte o Model Context Protocol. Selecione sua ferramenta para ver as instruções:

<Tabs>
  <Tab title="Claude Code">
    Execute o comando abaixo no terminal para adicionar o servidor MCP da Vendaze:

    ```bash
    claude mcp add --transport http vendaze https://mcp.vendaze.com/mcp
    ```

    O servidor ficará disponível em todas as sessões do Claude Code a partir desse momento.

  </Tab>
  <Tab title="Cursor">
    Abra o arquivo `~/.cursor/mcp.json` (ou crie-o se não existir) e adicione a entrada da Vendaze:

    ```json
    {
      "mcpServers": {
        "vendaze": {
          "url": "https://mcp.vendaze.com/mcp"
        }
      }
    }
    ```

    Reinicie o Cursor após salvar o arquivo. O servidor aparecerá no painel MCP das configurações.

  </Tab>
  <Tab title="Windsurf">
    Abra o arquivo `~/.codeium/windsurf/mcp_config.json` (ou crie-o se não existir) e adicione a entrada da Vendaze:

    ```json
    {
      "mcpServers": {
        "vendaze": {
          "serverUrl": "https://mcp.vendaze.com/mcp"
        }
      }
    }
    ```

    Reinicie o Windsurf após salvar. O servidor aparecerá na aba MCP das configurações.

  </Tab>
  <Tab title="Kiro AI">
    Abra o arquivo `.kiro/settings/mcp.json` na raiz do seu projeto (ou em `~/.kiro/settings/mcp.json` para configuração global) e adicione a entrada da Vendaze:

    ```json
    {
      "mcpServers": {
        "vendaze": {
          "url": "https://mcp.vendaze.com/mcp"
        }
      }
    }
    ```

    O Kiro reconecta automaticamente ao salvar o arquivo de configuração.

  </Tab>
  <Tab title="GitHub Copilot">
    Crie ou edite o arquivo `.vscode/mcp.json` na raiz do seu projeto e adicione a entrada da Vendaze:

    ```json
    {
      "servers": {
        "vendaze": {
          "url": "https://mcp.vendaze.com/mcp"
        }
      }
    }
    ```

    Para configuração global, acesse **Configurações > MCP Servers** no VS Code e adicione o servidor pela interface.

  </Tab>
  <Tab title="Gemini CLI">
    Abra o arquivo `~/.gemini/settings.json` (ou crie-o se não existir) e adicione a entrada da Vendaze:

    ```json
    {
      "mcpServers": {
        "vendaze": {
          "httpUrl": "https://mcp.vendaze.com/mcp"
        }
      }
    }
    ```

    O Gemini CLI carrega as configurações automaticamente no próximo uso.

  </Tab>
  <Tab title="Codex CLI">
    Abra o arquivo `~/.codex/config.json` (ou crie-o se não existir) e adicione a entrada da Vendaze:

    ```json
    {
      "mcpServers": {
        "vendaze": {
          "url": "https://mcp.vendaze.com/mcp",
          "transport": "http"
        }
      }
    }
    ```

    O Codex CLI carrega as configurações automaticamente no próximo uso.

  </Tab>
</Tabs>

## O que seu assistente pode fazer

Uma vez conectado, seu assistente de IA tem acesso às seguintes ações na sua conta Vendaze:

### People

| Ferramenta      | Descrição                                                |
| --------------- | -------------------------------------------------------- |
| `list_people`   | Lista contatos com suporte a filtros e paginação.        |
| `get_person`    | Busca uma pessoa por ID com campos adicionais e empresa. |
| `create_person` | Cria um novo contato no workspace.                       |
| `update_person` | Atualiza campos de um contato existente.                 |
| `delete_person` | Remove (soft delete) um contato.                         |

### Companies

| Ferramenta       | Descrição                                         |
| ---------------- | ------------------------------------------------- |
| `list_companies` | Lista empresas com suporte a filtros e paginação. |
| `get_company`    | Busca uma empresa por ID com campos adicionais.   |
| `create_company` | Cria uma nova empresa no workspace.               |
| `update_company` | Atualiza campos de uma empresa existente.         |
| `delete_company` | Remove (soft delete) uma empresa.                 |

### Deals

| Ferramenta    | Descrição                                                       |
| ------------- | --------------------------------------------------------------- |
| `list_deals`  | Lista negócios com suporte a filtros de pipeline, preço e data. |
| `get_deal`    | Busca um negócio por ID com pipeline, stage e produtos.         |
| `create_deal` | Cria um novo negócio.                                           |
| `update_deal` | Atualiza campos de um negócio existente.                        |
| `delete_deal` | Remove (soft delete) um negócio.                                |

### Tasks

| Ferramenta         | Descrição                                                   |
| ------------------ | ----------------------------------------------------------- |
| `list_tasks`       | Lista tarefas filtradas por entidade, prioridade ou status. |
| `create_task`      | Cria uma nova tarefa.                                       |
| `update_task`      | Atualiza uma tarefa existente.                              |
| `delete_task`      | Remove uma tarefa.                                          |
| `list_task_types`  | Lista os tipos de tarefa disponíveis no workspace.          |
| `create_task_type` | Cria um novo tipo de tarefa.                                |
| `update_task_type` | Atualiza um tipo de tarefa existente.                       |
| `delete_task_type` | Remove um tipo de tarefa.                                   |

### Activities

| Ferramenta        | Descrição                                                      |
| ----------------- | -------------------------------------------------------------- |
| `list_activities` | Lista atividades vinculadas a uma pessoa, empresa ou negócio.  |
| `get_activity`    | Busca uma atividade por ID.                                    |
| `create_activity` | Registra uma nova atividade (ligação, reunião, email ou nota). |
| `delete_activity` | Remove uma atividade.                                          |

### Products

| Ferramenta       | Descrição                                   |
| ---------------- | ------------------------------------------- |
| `list_products`  | Lista os produtos disponíveis no workspace. |
| `get_product`    | Busca um produto por ID.                    |
| `create_product` | Cria um novo produto.                       |
| `update_product` | Atualiza campos de um produto existente.    |
| `delete_product` | Remove (soft delete) um produto.            |

### Tags

| Ferramenta   | Descrição                                  |
| ------------ | ------------------------------------------ |
| `list_tags`  | Lista todas as tags do workspace.          |
| `get_tag`    | Busca uma tag por ID ou chave.             |
| `create_tag` | Cria uma nova tag.                         |
| `update_tag` | Atualiza nome ou cor de uma tag existente. |
| `delete_tag` | Remove uma tag do workspace.               |

### Lists

| Ferramenta    | Descrição                                 |
| ------------- | ----------------------------------------- |
| `list_lists`  | Lista as listas de contatos do workspace. |
| `create_list` | Cria uma nova lista de contatos.          |
| `update_list` | Renomeia uma lista existente.             |
| `delete_list` | Remove uma lista de contatos.             |

### Custom Fields

| Ferramenta            | Descrição                                            |
| --------------------- | ---------------------------------------------------- |
| `list_custom_fields`  | Lista os campos adicionais disponíveis no workspace. |
| `create_custom_field` | Cria um novo campo adicional.                        |
| `update_custom_field` | Atualiza um campo adicional existente.               |
| `delete_custom_field` | Remove um campo adicional.                           |

### Pipelines

| Ferramenta        | Descrição                                          |
| ----------------- | -------------------------------------------------- |
| `list_pipelines`  | Lista os pipelines e seus stages.                  |
| `get_pipeline`    | Busca um pipeline por ID com todos os seus stages. |
| `create_pipeline` | Cria um novo pipeline com stages.                  |
| `update_pipeline` | Atualiza um pipeline ou seus stages.               |

## Exemplos de perguntas

Depois de conectar, você pode conversar com o assistente de forma natural:

- "Quais são os 10 negócios com maior valor no pipeline Comercial?"
- "Mostre os contatos da empresa Acme Corp com todas as informações."
- "Quais tarefas estão abertas com prazo vencido?"
- "Crie um contato chamado João Silva, da empresa XYZ, com email joao@xyz.com."
- "Mova o negócio 'Contrato Trimestral' para o stage 'Proposta Enviada'."
- "Adicione uma nota no contato Ana Costa dizendo que ela retorna no dia 15."
- "Registra uma ligação no negócio da Acme Corp dizendo que estamos aguardando aprovação do jurídico."

## Observações

- O MCP Server aplica os mesmos rate limits da API REST.
- A conexão usa a autorização OAuth da sua conta Vendaze. O assistente acessa apenas o workspace que você autorizou.
- Campos adicionais são retornados apenas em ferramentas de busca por ID (`get_*`), não em listagens.
