## O que é a Vendaze API

A Vendaze API é uma interface REST pública que permite que aplicativos externos leiam e escrevam dados na Vendaze, o CRM B2B para times de vendas.

Com ela, você pode criar integrações nativas, automações, painéis customizados e agentes de IA que operam diretamente sobre os dados do seu time comercial.

## O que você precisa antes de começar

1. Uma conta na Vendaze com pelo menos um workspace ativo
2. Um servidor com suporte a HTTPS para receber o callback OAuth
3. Familiaridade básica com APIs REST e o fluxo OAuth 2.1

Não há processo de aprovação para se tornar um integrador. Qualquer desenvolvedor pode registrar um aplicativo e começar a construir imediatamente.
