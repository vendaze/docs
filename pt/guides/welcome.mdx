## O que é a Vendaze API

A Vendaze API é uma interface REST pública que permite que aplicativos externos leiam e escrevam dados no Vendaze, o CRM B2B para times de vendas.

Com ela, você pode criar integrações nativas, automações, painéis customizados e agentes de IA que operam diretamente sobre os dados do seu time comercial.

## Como a API funciona

O Vendaze é uma plataforma **multi-workspace**. Cada usuário pode pertencer a vários workspaces — como um consultor que atende vários clientes. Cada workspace é um ambiente completamente isolado com seus próprios dados: contatos, negócios, tarefas e configurações.

Quando um usuário autoriza seu aplicativo via OAuth, ele escolhe qual workspace conectar. Todas as chamadas de API feitas com aquele token operam exclusivamente naquele workspace. Você nunca precisa enviar ou rastrear o `workspace_id`. Ele é embutido no token de acesso e resolvido automaticamente pela API.

## O que você precisa antes de começar

1. Uma conta no Vendaze com pelo menos um workspace ativo
2. Um servidor com suporte a HTTPS para receber o callback OAuth
3. Familiaridade básica com APIs REST e o fluxo OAuth 2.1

Não há processo de aprovação para se tornar um integrador. Qualquer desenvolvedor pode registrar um aplicativo e começar a construir imediatamente.
