## O que você pode construir

A Vendaze API é uma interface REST que permite que aplicativos externos leiam e escrevam dados em workspaces da Vendaze. Ela usa OAuth 2.1 para autorização: seus usuários conectam o próprio workspace ao seu app e você opera sobre os dados deles com consentimento explícito.

Integrações típicas:

- **Sincronização nativa de CRM:** mantenha contatos, empresas e negócios sincronizados entre a Vendaze e seu produto
- **Automação de vendas:** crie tarefas, registre atividades e mova negócios no pipeline com base em eventos do seu sistema
- **Painéis customizados:** extraia dados do workspace para ferramentas internas, plataformas de BI ou data warehouses
- **Agentes de IA:** conecte modelos de linguagem a dados reais do CRM via MCP server

## Como funciona

A API usa **OAuth 2.1 Authorization Code flow**. Seu app se registra uma vez para obter um `client_id` e `client_secret`. Quando um usuário quer conectar o workspace, ele passa por uma tela de consentimento OAuth padrão. Após a autorização, você recebe um `access_token` com escopo para aquele workspace específico.

Toda chamada à API carrega esse token. O contexto do workspace é resolvido automaticamente. Você nunca envia um identificador de workspace nas requisições.

## O que você precisa para começar

- Uma conta na Vendaze com pelo menos um workspace ativo ([crie gratuitamente](https://app.vendaze.com/register))
- Um servidor com suporte a HTTPS para receber o callback OAuth
- Familiaridade com APIs REST e OAuth 2.1

Não há processo de aprovação. Qualquer desenvolvedor pode registrar um app e começar a construir imediatamente.
